Steps to Setup a Virtual Environment on Windows using PowerShell:

    Step 01: cd into desired folder.
    Step 02: mkdir needed project directory.
    Step 03: Check pip version> python -m pip list
    Step 04: Update if necessary> python.exe -m pip install --upgrade pip
    Step 05: Check installed packages> python -m pip list
    Step 06: Check if the virtual envronment was created properly> ls
    Step 07: If running scripts is disabled on your system> Set-ExecutionPolicy RemoteSigned
    Step 08: Activate the virtual environment> .\virtualenv\Scripts\Activate.ps1
    Step 09: Launch VS Code in the current directory> code .
    Step 10: Install Django> pip install Django
    Step 11: Deactivate Django> deactivate
